About

Hello! My name is Sebastian Alejandro and I specialize in Javascript. If I'm not learning the web's latest and greatest technologies, you can find me building and flying FPV racing drones.

I have a diverse set of skills from content-focused image and video editing with the Adobe Creative Cloud, to programming-intensive Javascript implementations using React Express and Node. I am especially interested in educational technologies, flying robots, and political philosophy.

I primarily use Javascript, but picking up a new framework of language isn't a problem. From React to Angular, Express to Node, this is a collection of some of the projects that best embody my skills.
